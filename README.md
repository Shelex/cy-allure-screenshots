## Example how to add screenshots to allure-results

### Pre-requisites
 - [Allure](https://docs.qameta.io/allure/#_get_started)
 - Other dependencies with `yarn install`

### How to execute:
 - `yarn cy:run`
 - `yarn allure:attachments`
 - `yarn allure:report`

### Open generated report
 - `allure open`
